# Summary

I have my 3d printer only for a few weeks now.
This is one of my first designs. 
Newbie at work :-)

Windchimes dupont wires holder.

Dupont wires are often used with Arduino

Dupont wires can create a mess, a labyrinth of many many wires.
Try to find the right wires you want, among hundreds of wires.

This Dupont Wires Holder makes it easier to find the right dupont wire.

The wires can be sorted on pin combination, pin-pin, pin-bus, bus-bus.
And can be sorted on color.

There are parts to make holders for 10 cm, 20 cm and 30 cm dupont wires.


FREECAD file included.
