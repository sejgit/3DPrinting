# Summary

I need display on my new Prusa i3 MK3 mounted on side. My old printing enclosure IKEA Stuva isn't deep enough for this printer. Unluckily printer was completed when I found this fail. I'm very lazy person so part is designed as mountable on finished printer. It could be designed better way / more sturdy for mouting during build, but this wasn't my case.
