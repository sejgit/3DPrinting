# Summary

These are a selection of float brackets for [these float switches](https://amzn.to/2Bedkux). Different sizes for different rim / depth setups. I use them as the auto-top-off sensor in my [reef-pi](https://reef-pi.github.io/) aquarium build.

The ones with two "shelves" can be used with one switch up and one down to provide an overflow backup sensor.

Print in PETG if you are putting them in any container with fish!
