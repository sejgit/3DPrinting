# Summary

I wasn't happy with the available Mk3 knobs, and I wanted to test a design for the MMU2 so I created this knob inspired by the SpaceX landing barge design.  In my opinion, it looks best when printed with the color against the PEI sheet (my first layers always look better than my top layers).  I printed one each way, pictures of both are attached.  Print will work best with a 0.4mm nozzle.
