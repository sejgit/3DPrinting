# Summary

The original design caused the batteries to jam and not exit the box on the ramp. I solved this by adding a single ramp to the cover. 

Only AA version for the time being. While still work in progress, this is the first working prototype which does the job quite well so far. Print face down so it doesn't need any supports. Including .c4d file.
