# Summary

If you remove the PSU from the frame of the Original Prusa i3 MK3 (for example because you want to have the PSU outside of an enclosure) you can replace it with this brace to keep the frame rigid.
You can see it's usage in this video:
https://www.youtube.com/watch?v=oS7ZtBNh2hE

Print from a rigid (so not flexible) material.
PLA will work just fine.
